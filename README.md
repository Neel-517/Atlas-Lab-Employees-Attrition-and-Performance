# Atlas Lab Employees Attrition and Performance

## [Tableau Dashboard](https://public.tableau.com/views/AtlasLabemployees/Demographic?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

In order to spot trends of low satisfaction and direct HR strategy, this analysis looks at Atlas Lab's employee happiness indicators, which include environment, job, work-life balance, and relationships. Calculating the percentage of workers who gave a score of 1 or 2 on a 5-point scale for each category using SQL. The goal was to identify trouble spots that might be linked to the risk of burnout and turnover.
